## Getting started

**Install dependencies**

```sh
yarn install
```

**Run dev server**

```sh
yarn dev
```

**Build app**

```sh
yarn build
# Deploy the static app with Netlify / Surge.sh / Zeit
```

Update the env var `REACT_APP_GITHUB_TOKEN` (in `.env`) with your GitHub token (for higher rate limit)
